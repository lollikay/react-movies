# Incredible Web

[Incredible Web ](https://www.incredible-web.com/) constantly innovates its design processes and implements the best technologies available to guarantee an optimal solution.

## Installation

### Install packages

We recommend using [Node Latest LTS Version ](https://nodejs.org/en/download/).

```sh
npm install
```

### Running the app

To run the app in development mode:

```bash
npm run dev
```
